LSTM_航班预测

发表于 2018-07-24 | 分类于 深度学习DL | | 阅读次数:

一、认识

1.1简述

  LSTM是改进型的循环神经网络，每一个神经元的输入包括上一个时刻的状态$C{t-1}$、上一个时刻的输出$h_{t-1}$与当前时刻的输入$X_t$。神经元内部包括三个门：
  遗忘门$f_t$：生成遗忘向量，和t-1时刻的状态$ C_{t-1} $进行卷积，结果用于产生当前状态$ C_t $。
  输入门$i_t$：生成输入向量，和t-1时刻的输出值$ h_{t-1} $、当前时刻的输入$ x_t $作用，结果用于产生当前状态$C_t$。
  输出门$o_t$：生成输出向量，和当前时刻的状态$ C_t $作用，结果作为输出值$h_t$影响下一个时刻的状态。

1.2关键点
  • 类似数电里面的时序逻辑电路，当前时刻的状态由当前的输入已经之前的状态共同决定。
1.3作用及应用
  • 处理序列化的问题，预测

四、Future

  1. 精彩点与启发

    • LSTM是循环神经网络(RNN)的改进，针对RNN中，记忆序列过长时梯度消失和梯度爆炸的问题，LSTM通过遗忘门、输入门的机制予以改善，但没有实质上解决问题。
    • 循环神经网络通过“一维循环”机制，使得一个神经元能够充当无数个神经元，能否增加循环线路，变成多维的循环，使得神经网络的物理结构简化而逻辑结构更加复杂？

  2. 设想与工作

SVM

发表于 2018-07-23 | 分类于 机器学习ML | | 阅读次数:

一、认识

1.1简述
1. SVM是一种分类算法，输入样本的各项指标和标签，经过学习之后根据未知样本的标签指标预测其标签。

2. 对于线性分类问题：通过对样本及其标签的学习，得到分类超平面，以此对未知样本进行分类。

3. 对于非线性分类问题：使用映射函数将原样本映射到另一个空间，使得映射之后线性可分。
1.2关键点
1. 使用超平面对样本进行分类。
2. 使用变换函数将样本从低维映射到高维，实现非线性样本的分类。
3. 用拉格朗日乘子法，解二次优化问题，求得最优参数。
无论是在低维空间还是高维空间中解决分类问题，都需要计算向量內积，核函数的作用就是：利用低维空间的內积计算高维空间的內积。我的理解就是：将低维空间的內积经函数映射，变成高维空间的內积。
4. 在SVM模型中，最终需要的只是向量內积，可以不需要映射函数，但映射之后的內积需要有。核函数可以直接计算高维空间的內积而不用管映射函数，并且在传统方法中，使用映射函数映射后再计算內积的计算代价高，因此核函数能够简化计算。
